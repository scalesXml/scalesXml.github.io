scales-sbt-plugins 0.2.2 API

This document is the API specification for scales-sbt-plugins 0.2.2 API

Class Summary
trait ArtifactsHelper extends sbt.BasicManagedProject
case class Copy (val from : java.lang.String, val to : java.lang.String) extends scala.Product
trait DocWrapper extends SxrPaths with sbt.MavenStyleScalaPaths
Logic for replacing source code links in scaladocs with sxr code.
case class MarkupHeader (val title : java.lang.String, val extraHeaders : java.lang.String, val overrideHeader : Boolean) extends scala.Product
Allows providing an additional header set per individual markup. Title gets special treatment as it is likely the most common case, given the siteHeaders. If the user really only wants a certain set of headers then they can set overrideHeaders to true.
case class Site (val copy : Copy, val siteDesc : java.lang.String, val siteLink : scala.Option[java.lang.String], val ignore : scala.Iterable[sbt.FileFilter]) extends scala.Product
siteDesc is a mediawiki snippet (will simply be embedded) to represent the part of the site that copy refers to. siteLink is optionally the target html to load for that link, defaulting to copy.to / index.html. It MUST be in utf-8 format The ignore list can be tailored for just a site, it will be added to the siteIgnore list upon running site.
trait SiteBase extends AnyRef
trait SiteProject extends DocWrapper with SiteBase with ArtifactsHelper
Utilities and tasks for creating a site
trait SxrHelper extends SxrPaths with sbt.AutoCompilerPlugins with sbt.MavenStyleScalaPaths
Configures sxr to: 1) disable during consoles 2) configure correct linking from test cases into main src 3) ensure the correct base directory is used
trait SxrLineWrapper extends SxrPaths
Wraps the output of sxr to add line number anchors. Sxr uses newlines in a very large pre block so this filters on newlines
trait SxrPaths extends sbt.BasicScalaProject
trait SxrRelativeTestUrls extends SxrHelper
Replace the main source links with relative paths Basically looks for ... where the path starts with mainSxrPath as a url with the relative paths.
trait SxrWrapper extends SxrLineWrapper with SxrRelativeTestUrls
Provides simple sxr helping, relative path handling for test code and line wrapping for sxr output
trait TestReview extends SiteBase
class TestReviewListener (links : scala.List[(java.io.File, java.lang.String)], project : sbt.Project) extends sbt.TestsListener with AnyRef
Creates a mediawiki report for displaying test results. Both classes and stack contents are matched against sxr output.
trait TestReviewer extends TestReview with SxrHelper
Links directly to SxrWrappers as well, which may not be wanted. If its not wanted just use TestReview instead and provide your own testListeners function
